# Welcome to the CANOE project! 

We utilize the Temoa framework to investigate decarbonization and policy strategies for Canada. 

To access the dataset please visit: [CANOE dataset](https://zenodo.org/records/18435395)

To access the documentation: [CANOE Documentation](https://canoe-main.github.io/CANOE/)

To view the people involved in the project, please visit: [CANOE group](https://sustainablesystems.civmin.utoronto.ca/canadian-open-energy-canoe-model/)

## Getting started with CANOE

**1)** Check the [Youtube videos](https://www.youtube.com/playlist?list=PLQwQoDG4eHizZf0yTVr81mNoCQDWXWm_c) and [Tutorial](https://github.com/CANOE-main/Onboarding)

**2)** Download [Temoa](https://github.com/CANOE-main/temoa) (clone the CANOE-main/Temoa)

**3)** Download the [dataset](https://zenodo.org/records/18435395) from Zenodo

**4)** Clone the [Interface repository](https://github.com/CANOE-main/canoe_interface) 

**5)** Run the Interface to filter for province and resolution; Sectors; Scenarios; 

**6)** Clone and run the [Representative Periods](https://github.com/CANOE-main/representative_periods)

**7)** Take the output from the rep periods and run Canoe 

  **7a)** Change the config file to match the database they want to use
  
  **7b)** Set the Gurobi or equivalent solver
  
**8)** Troubleshooting: 

  **8a)** Check the commodities graph
  
  **8b)** Check the log 
  
**9)** Check the [FAQ](https://canoe-main.github.io/CANOE/faqs/)

**10)** [Contact us](https://sustainablesystems.civmin.utoronto.ca/canadian-open-energy-canoe-model/) 

**11)** [Documentation](https://canoe-main.github.io/CANOE/)
